# 성신컴퍼니 통관비 계산 로직

## 📋 개요

이 문서는 성신컴퍼니 통관비 계산 툴의 전체 계산 로직을 설명합니다. CIF 기준으로 관세와 부가세를 계산하며, 배송비는 0.5kg 단위로 올림 처리됩니다.

---

## 1️⃣ 배송비 계산 로직 (과세 운임)

### 기본 원칙

- **0.5kg 단위 올림 처리**: 입력된 무게를 0.5kg 단위로 올림하여 계산
- **계단식 요금제**: 0.5kg 단위로 배송비가 증가
- **기본 배송비**: 0.5kg 기준 5,800원
- **추가 배송비**: 0.5kg 초과분당 1,000원

### 계산 공식

```javascript
calculateShippingCost(weight: number): number
```

### 단계별 계산 과정

#### 1단계: 입력값 검증
```javascript
if (weight <= 0) return 0;
```

#### 2단계: 0.5kg 단위 올림 처리
```javascript
const roundedWeight = Math.ceil(weight * 2) / 2;
```

**올림 예시:**
- 0.1kg → 0.5kg
- 0.6kg → 1.0kg
- 1.2kg → 1.5kg
- 1.5kg → 1.5kg (유지)
- 2.0kg → 2.0kg (유지)
- 2.3kg → 2.5kg

#### 3단계: 배송비 계산
```javascript
const baseCost = 5800;  // 기본 배송비 (0.5kg 기준)
const additionalCost = (roundedWeight - 0.5) * 1000;  // 0.5kg당 1,000원
return baseCost + Math.max(0, additionalCost);
```

### 배송비 표

| 무게 범위 (kg) | 올림된 무게 (kg) | 계산식 | 배송비 (원) |
|---------------|----------------|--------|------------|
| 0.1 ~ 0.5 | 0.5 | 5,800 + (0.5 - 0.5) × 1,000 | **5,800** |
| 0.6 ~ 1.0 | 1.0 | 5,800 + (1.0 - 0.5) × 1,000 | **6,300** |
| 1.1 ~ 1.5 | 1.5 | 5,800 + (1.5 - 0.5) × 1,000 | **6,800** |
| 1.6 ~ 2.0 | 2.0 | 5,800 + (2.0 - 0.5) × 1,000 | **7,300** |
| 2.1 ~ 2.5 | 2.5 | 5,800 + (2.5 - 0.5) × 1,000 | **7,800** |
| 2.6 ~ 3.0 | 3.0 | 5,800 + (3.0 - 0.5) × 1,000 | **8,300** |

**공식**: 배송비 = 5,800 + (올림된 무게 - 0.5) × 1,000

## 2️⃣ 세금 계산 로직 (CIF 기준)

### CIF란?

**CIF (Cost + Insurance + Freight)**
- **Cost**: 물건 가격 (매입 원가 × 환율)
- **Insurance**: 보험료 (물건 가격의 0.75%)
- **Freight**: 과세 운임 (배송비)

### 계산 흐름도

```
입력값
├─ 매입 원가 (¥)
├─ 적용 환율 (₩)
├─ 예상 무게 (kg)
└─ 카테고리 (관세율, 부가세율)

↓

1단계: 물건 가격 계산 (Cost)
물건 가격 = 매입 원가 × 적용 환율

↓

2단계: 배송비 계산 (Freight)
배송비 = calculateShippingCost(예상 무게)

↓

3단계: 보험료 계산 (Insurance)
보험료 = 물건 가격 × 0.0075 (0.75%)

↓

4단계: 과세표준 계산 (CIF 기준)
과세표준 = 물건 가격 + 배송비 + 보험료

↓

5단계: 관세 계산
관세 원본 = 과세표준 × 관세율
관세 = Math.floor(관세 원본 / 10) × 10  (10원 단위 내림)

↓

6단계: 부가세 계산
부가세 원본 = (과세표준 + 관세) × 부가세율
부가세 = Math.floor(부가세 원본 / 10) × 10  (10원 단위 내림)

↓

7단계: 총 납부 세액
총 납부 세액 = 관세 + 부가세
```

### 계산 함수

```javascript
calculateTax(
  purchasePrice: number,    // 매입 원가 (위안)
  exchangeRate: number,      // 적용 환율 (원)
  weight: number,            // 예상 무게 (kg)
  dutyRate: number,          // 관세율 (카테고리별)
  vatRate: number            // 부가세율 (0.10 = 10%)
)
```

### 상세 계산 코드

```javascript
// 1. 물건 가격 (Cost)
const itemPrice = purchasePrice * exchangeRate;

// 2. 배송비 (Freight)
const shippingCost = calculateShippingCost(weight);

// 3. 보험료 (Insurance)
const insurance = itemPrice * 0.0075;  // 0.75%

// 4. 과세표준 (CIF 기준)
const taxableBase = itemPrice + shippingCost + insurance;

// 5. 관세 계산 (10원 단위 내림)
const dutyRaw = taxableBase * dutyRate;
const duty = Math.floor(dutyRaw / 10) * 10;

// 6. 부가세 계산 (10원 단위 내림)
const vatRaw = (taxableBase + duty) * vatRate;
const vat = Math.floor(vatRaw / 10) * 10;

// 7. 총 납부 세액
const totalTax = duty + vat;
```

## 3️⃣ 계산 예시

### 예시 1: 의류 (관세율 13%, 부가세율 10%)

**입력값:**
- 매입 원가: 100위안
- 적용 환율: 190원
- 예상 무게: 1.2kg
- 카테고리: 일반 의류 (관세율 13%)

**계산 과정:**

1. **물건 가격** = 100 × 190 = **19,000원**

2. **올림된 무게** = 1.5kg

3. **배송비** = 5,800 + (1.5 - 0.5) × 1,000 = **6,800원**

4. **보험료** = 19,000 × 0.0075 = **142.5원** → **143원** (반올림)

5. **과세표준** = 19,000 + 6,800 + 143 = **25,943원**

6. **관세 원본** = 25,943 × 0.13 = 3,372.59원
   - **관세** = Math.floor(3,372.59 / 10) × 10 = **3,370원** (10원 단위 내림)

7. **부가세 원본** = (25,943 + 3,370) × 0.10 = 2,931.3원
   - **부가세** = Math.floor(2,931.3 / 10) × 10 = **2,930원** (10원 단위 내림)

8. **총 납부 세액** = 3,370 + 2,930 = **6,300원**

### 예시 2: 가전제품 (관세율 0%, 부가세율 10%)

**입력값:**
- 매입 원가: 500위안
- 적용 환율: 190원
- 예상 무게: 2.3kg
- 카테고리: 컴퓨터/노트북/태블릿 (관세율 0%)

**계산 과정:**

1. **물건 가격** = 500 × 190 = **95,000원**

2. **올림된 무게** = 2.5kg

3. **배송비** = 5,800 + (2.5 - 0.5) × 1,000 = **7,800원**

4. **보험료** = 95,000 × 0.0075 = **712.5원** → **713원** (반올림)

5. **과세표준** = 95,000 + 7,800 + 713 = **103,513원**

6. **관세** = 103,513 × 0.00 = **0원**

7. **부가세 원본** = (103,513 + 0) × 0.10 = 10,351.3원
   - **부가세** = Math.floor(10,351.3 / 10) × 10 = **10,350원** (10원 단위 내림)

8. **총 납부 세액** = 0 + 10,350 = **10,350원**

## 5️⃣ 중요 사항

### 5.1 CIF 기준 계산

- **CIF (Cost, Insurance, Freight)**: 관세청의 세금 계산 기준
- **과세표준** = 물건 가격 + 배송비 + 보험료
- 배송비와 보험료가 세금 계산에 포함됨

### 5.2 무게의 중요성

- 무게가 증가할수록 배송비가 증가
- 배송비가 증가하면 과세표준이 증가
- 과세표준이 증가하면 관세와 부가세도 증가
- **결론**: 무게를 줄이면 세금도 줄어듭니다

### 5.3 계산 정확도

- **무게**: 0.5kg 단위로 올림 처리
- **배송비**: 정수로 계산
- **보험료**: 물건 가격의 0.75% (소수점 발생 가능)
- **관세**: 10원 단위로 내림 처리
- **부가세**: 10원 단위로 내림 처리
- **최종 금액**: 정수로 표시

### 5.4 보험료 계산

- 보험료는 물건 가격의 **0.75%**로 계산
- 실제 통관 시 관세청이 산정하는 통상 보험료가 과세표준에 합산될 수 있음
- 본 계산은 예상 보험료를 포함한 금액임

## 7️⃣ 참고사항

### 7.1 실제 통관 시 주의사항

- 본 계산은 보험료가 포함된 금액이며, 실제 통관 시 관세청이 산정하는 통상 보험료가 과세표준에 합산될 수 있습니다.
- 사업자 통관(목록통관 배제)은 금액과 상관없이 관부가세가 발생합니다.
- 부피가 큰 상품은 실중량 대신 부피 무게(가로×세로×높이/6000)로 배송비가 적용될 수 있습니다.
- 골프채 등 고급 소비재는 수입가가 높을 경우 특소세(개별소비세/교육세)가 추가로 발생할 수 있습니다.
- 식품류는 식품검역 비용(건당 약 10만 원)이 별도로 발생할 수 있습니다.

### 7.2 계산 기준

- 이 계산 로직은 성신컴퍼니의 해운 배송비 기준을 따릅니다.
- 실제 통관 시에는 추가 비용이 발생할 수 있습니다.
- 환율 변동에 따라 계산 결과가 달라질 수 있습니다.
- 카테고리별 관세율은 실제 관세청 기준을 반영합니다.

## 8️⃣ 계산 공식 요약

### 배송비 계산
```
올림된 무게 = Math.ceil(입력 무게 × 2) / 2
배송비 = 5,800 + (올림된 무게 - 0.5) × 1,000
```

### 세금 계산
```
물건 가격 = 매입 원가 × 적용 환율
배송비 = calculateShippingCost(예상 무게)
보험료 = 물건 가격 × 0.0075
과세표준 = 물건 가격 + 배송비 + 보험료
관세 = Math.floor(과세표준 × 관세율 / 10) × 10
부가세 = Math.floor((과세표준 + 관세) × 부가세율 / 10) × 10
총 납부 세액 = 관세 + 부가세
```
