# 라이브 히어로 A/B 카피안

## 공통 방향
- 1차 목적: 관심 유도(스크롤/체류)
- 2차 목적: 상담 전환
- 톤: 프리미엄 B2B, 실행력/투명성 강조

## Variant A (기준안)

- 배지: `LIVE 운영중`
- 헤드라인:
  - `중국 무역 현장을`
  - `실시간으로 공개합니다`
- 본문:
  - `경쟁사와 달리 실제 운영 화면을 공개해 작업 투명성을 보여드립니다. 상담 전, 운영 역량부터 직접 확인해 보세요.`
- 의도:
  - 차별점과 투명성 중심의 안정형 메시지

## Variant B (실험안)

- 배지: `실시간 운영 공개`
- 헤드라인:
  - `보여주기식 소개 대신`
  - `실제 운영 현장을 공개합니다`
- 본문:
  - `24시간 운영 화면으로 검수, 포장, 출고 흐름을 투명하게 보여드립니다. 광고 문구보다 실행력을 먼저 확인하세요.`
- 의도:
  - 강한 문제 제기로 임팩트를 높이는 도전형 메시지

## 코드 적용 방식

- 환경변수로 제어:
  - `VITE_LIVE_HERO_VARIANT=A` (기본)
  - `VITE_LIVE_HERO_VARIANT=B` (실험)
- 적용 파일:
  - `components/LiveStream.tsx`

## 측정 권장값

- 비교 지표:
  - `live_hero_view -> cta_click` 전환율
  - 스크롤 50% 도달률
  - 외부 링크 클릭률(낮을수록 내부 전환 우세)
- 최소 샘플:
  - 각 변형당 동일 광고 조건에서 500세션 이상 권장
